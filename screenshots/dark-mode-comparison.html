Decision

Option A vs Option B

Both replace the current navy dark mode. Pick one.

Option A: Pure Black

  • True OLED black — maximum battery savings on AMOLED
  • Matches the reference screenshot you shared exactly
  • Crisp card borders (#262626) prevent cards from merging with background
  • Brighter accent (#2dd4bf vs #0d9488) compensates for max contrast
  • Neutral gray tones (zinc scale) instead of bluish slate — feels more "tech"
  • Hero gradient darker (teal-950) — subtle teal glow, not navy
  • Similar to: X (Twitter) OLED, Reddit OLED, Cash App dark

Option B: Elevated Black

  • True black background + slightly elevated cards (#111) for depth
  • Soft translucent borders (white 6% opacity) instead of hard lines
  • Subtle card shadows add layering depth on non-OLED screens
  • Same accent as light mode (#14b8a6) — more brand-consistent
  • iOS-inspired elevation system (Apple uses #1c1c1e for grouped cards)
  • Feels warmer than Option A while still being true black
  • Similar to: Apple iOS dark, Revolut dark, Notion dark

Implementation scope for either option:

4 files — global.css (CSS vars) + tokens.ts (dark colors) + gradients.ts (hero gradients) + Card component (conditional border)
If adding as 4th theme option: also profile.tsx (theme selector) + lib/theme.ts (new preference)
